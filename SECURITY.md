# Security Policy

## Supported Versions

Only the current main branch (`master`) is actively updated with security patches.

| Version | Supported          |
| ------- | ------------------ |
| `master`| :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover any security-related flaw in our documentation, such as a malicious payload disguised as a legitimate snippet, or an accidental exposure of internal secrets, please **DO NOT** open a public issue.

Instead, please report it privately:

1. Email the maintainers directly at `info.digitalbulwark@gmail.com`.
2. Include the exact file name and line number of the suspicious code.
3. Briefly explain why the code is malicious or insecure.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to resolve the issue as quickly as possible. We value your efforts in keeping this Arsenal safe for everyone.
